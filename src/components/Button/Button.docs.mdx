<Description>
**Button** supports:
- Variants: <code>primary</code>, <code>secondary</code>, <code>success</code>, <code>danger</code>, <code>warning</code>, <code>neutral</code>, <code>outline</code>, <code>ghost</code>
- Sizes: <code>sm</code>, <code>md</code>, <code>lg</code>
- Tokens via <code>vars</code> (colors, radius, paddings, focus ring)
- Slot class overrides via <code>classNames</code>
- Loading state, left/right icons, and fullWidth
</Description>
